# Real Estate Price Prediction

Machine learning project predicting Chicago real estate prices using:
- Numeric features (ensemble model: R² = 0.70)
- Sentiment analysis (text descriptions: R² = 0.52)
- Smart fusion approach

## Files
- `numeric_analysis_code.py` - Numeric model
- `sentiment_analysis_code.py` - Sentiment model  
- `smart_fusion_code.py` - Fusion model
- `real_estate_data_chicago.csv` - Dataset


## Results
Meta Learner Fusion gives R^2 0.7230
